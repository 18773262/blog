## 代码中直接下载模型  
其实非常简单，只要在代码中加一行就行了  
```  
import os  
os.environ['HF_ENDPOINT'] = 'https://hf-mirror.com'  
```  
  
<b>注意</b>  
```  
os.environ['HF_ENDPOINT'] = 'https://hf-mirror.com'  
这段代码一定要放在  
from transformers import pipeline  
前面，不然就访问不到 ，应该是transformer会调用这个environment，不提前设置就不行  
```
